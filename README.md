# Приложение "Военная авторизация"

## Что это?
Десктопное приложение с графическим интерфейсом на Python (Tkinter), которое реализует систему авторизации и регистрации пользователей с проверкой сложности пароля. После успешного входа открывается окно с праздничным поздравлением в военно-патриотическом стиле, приуроченное к 23 февраля.

**Функционал:**
- Регистрация новых пользователей
- Вход в систему
- Проверка пароля на сложность (минимум 8 символов и наличие спецсимволов $, #, @)
- Отображение праздничного окна с изображением после авторизации
