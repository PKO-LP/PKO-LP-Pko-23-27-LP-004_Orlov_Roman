# Приложение "Новогодняя авторизация"

## Что это?
Десктопное приложение с графическим интерфейсом на Python (Tkinter), которое реализует систему авторизации и регистрации пользователей с проверкой сложности пароля. После успешного входа открывается праздничное окно с новогодним поздравлением. Приложение имеет стильный интерфейс с цветовой гаммой, имитирующей праздничный стиль. Функционал включает регистрацию новых пользователей с проверкой логина на уникальность, вход в систему существующих пользователей, проверку пароля на сложность (минимум 8 символов и наличие спецсимволов $, #, @), очистку полей ввода кнопкой "Сброс", отображение праздничного окна с изображением и поздравлением после авторизации, а также адаптивный интерфейс с центрированием окон на экране.

## Как клонировать проект
git clone https://github.com/Sement-000/auth-app.git
cd auth-app

## Зависимости
Для работы приложения необходимо установить библиотеку Pillow для работы с изображениями. Выполните команду: pip install Pillow. Tkinter обычно входит в стандартную поставку Python. Если его нет: на Windows он уже установлен, на Linux выполните sudo apt-get install python3-tk, на macOS выполните brew install python-tk.

## Как запустить
python tk.py

## Важно! Изображение для приложения
Приложение ищет файл 7.png в той же папке, где находится tk.py. Где взять изображение: скачайте любую новогоднюю картинку с unsplash.com или pixabay.com, сохраните её как 7.png в папку с проектом, или переименуйте ваше изображение в 7.png (формат PNG). Если изображение отсутствует, приложение автоматически покажет текстовое поздравление "С новым годом" без картинки. Интерфейс адаптируется и будет выглядеть празднично даже без изображения.

## Структура кода и пояснения
Основной класс приложения - AuthApp, который управляет всем функционалом. В методе __init__ происходит инициализация главного окна приложения, задается его размер 450x600 пикселей, запрещается изменение размера, окно центрируется на экране, создается словарь для хранения пользователей self.users = {} и вызывается метод создания интерфейса create_widgets. Метод center_window рассчитывает координаты для центрирования окна на экране: получает ширину и высоту экрана, вычисляет позицию по формулам x = (screen_width - width) // 2 и y = (screen_height - height) // 2, затем применяет геометрию с этими координатами.

Метод create_widgets создает графический интерфейс приложения: main_frame - основной контейнер с серым фоном и отступами, title_label - заголовок "АВТОРИЗАЦИЯ / РЕГИСТРАЦИЯ", form_frame - рамка для формы с белым фоном и обводкой, login_entry - поле ввода логина с фокусом при запуске, password_entry - поле ввода пароля где символы скрыты звездочками, btn_row1 - ряд с кнопками "Войти" синяя и "Регистрация" зеленая, btn_row2 - кнопка "Сброс" красная для очистки полей, requirements_frame - рамка с требованиями к паролю, status_label - строка статуса внизу "Ожидание ввода...", а также привязка клавиши Enter к функции входа.

Метод validate_password проверяет пароль на соответствие требованиям безопасности: сначала проверяет длину пароля минимум 8 символов, затем проверяет наличие хотя бы одного спецсимвола из набора ['$', '#', '@']. Возвращает кортеж с флагом успеха и сообщением. Метод login обрабатывает вход пользователя: получает логин и пароль из полей ввода, проверяет заполнение всех полей, проверяет существование пользователя в словаре self.users, при успехе очищает поля, скрывает главное окно через root.withdraw и вызывает show_congratulations, при неудаче показывает сообщение об ошибке.

Метод register обрабатывает регистрацию нового пользователя: получает логин и пароль из полей ввода, проверяет заполнение всех полей, проверяет уникальность логина, проверяет пароль методом validate_password, при успехе добавляет пользователя в словарь self.users[login] = password, показывает сообщение об успехе, затем очищает поля и обновляет статус. Метод clear_fields очищает поля ввода логина и пароля и устанавливает фокус на поле логина.

Метод show_congratulations создает праздничное окно после успешной авторизации: создает новое окно Toplevel с заголовком "С новым годом!", настраивает обработчик закрытия окна который закрывает всю программу, задает праздничную цветовую гамму с bg_color1 = "#2b4f2b" темно-зеленый для основного фона, accent_color = "#c0392b" красный для акцентов и gold_color = "#ffd700" золотой для декора. Затем проверяет наличие файла "7.png" через os.path.exists. Если файл найден, загружает изображение через Image.open, конвертирует для Tkinter через ImageTk.PhotoImage, рассчитывает размер окна под изображение, создает рамку с золотой окантовкой для фото и отображает изображение. Если файл не найден, показывает текстовое поздравление крупным шрифтом. Далее создает рамку "письмо с фронта" со стилизованным текстом: заголовок с логином пользователя, стихотворные строки поздравления "За окном снежинок стая, Желаем крепкого здоровья, Тоже водит хоровод. Попрощавшись с годом старым, Мы встречаем Новый год" и золотую звезду внизу. Добавляет кнопку "ЗАКРЫТЬ" в праздничном стиле.

В блоке запуска приложения if __name__ == "__main__": сначала проверяется наличие библиотеки Pillow через try except. Если библиотека не установлена, выводится понятная инструкция по установке "УСТАНОВИТЕ БИБЛИОТЕКУ Pillow: pip install Pillow" и программа завершается. При успешной проверке создается главное окно root = tk.Tk(), создается экземпляр приложения app = AuthApp(root) и запускается основной цикл root.mainloop.
