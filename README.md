# Приложение "Новогодняя авторизация"

## Что это?
Десктопное приложение с графическим интерфейсом на Python (Tkinter), которое реализует систему авторизации и регистрации пользователей с проверкой сложности пароля. После успешного входа открывается окно с праздничным поздравлением в новогоднем стиле.

**Функционал:**
- Регистрация новых пользователей с валидацией пароля
- Вход в систему
- Проверка пароля на сложность (минимум 8 символов и наличие спецсимволов $, #, @)
- Отображение праздничного окна с новогодним поздравлением и изображением
- Адаптивный интерфейс с центрированием окон
